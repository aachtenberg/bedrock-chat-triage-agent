AWS Bedrock

Knowledge Chat

Simple, low-ops interface for querying an S3-backed Bedrock Knowledge Base through CloudFront, API Gateway, and Python Lambdas.

CloudFront Basic Auth WAF Sonnet 4.5
Modes
  • Ask: retrieve context and generate an answer with citations.
  • Search: inspect the top matching passages before asking.
Notes

Knowledge-base ingestion is separate from Terraform state. Re-run ingestion after changing files under the knowledge-base folder.

Console

Ready
Start with a question about the uploaded documents, or switch to Search to inspect the retrieved passages directly.
The app calls the same CloudFront host under /api, so WAF and auth cover both the site and the backend.
